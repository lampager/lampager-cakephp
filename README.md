# Lampager for CakePHP

Rapid pagination without using OFFSET

## Requirements

- PHP: ^5.6 || ^7.0
- CakePHP: ^3.6
- [lampager/lampager](https://github.com/lampager/lampager): ^0.4

## Installing

```bash
composer require lampager/lampager-cakephp
```
